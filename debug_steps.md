# 게시글 로딩 문제 디버깅 가이드

## 즉시 확인할 것

### 1. 브라우저 콘솔 확인
1. F12 눌러서 개발자 도구 열기
2. Console 탭 선택
3. 에러 메시지 확인 (빨간색)
4. 스크린샷 찍어서 보내주세요!

### 2. Network 탭 확인
1. F12 → Network 탭
2. 페이지 새로고침
3. 실패한 요청 찾기 (빨간색)
4. 클릭해서 상세 정보 확인
5. 스크린샷 찍어서 보내주세요!

## 가능한 원인

### A. 시크릿 모드에서만 작동 = 캐시/쿠키 문제
- **해결책**: 브라우저 캐시 완전 삭제
  1. Ctrl + Shift + Delete
  2. "전체 기간" 선택
  3. "쿠키 및 기타 사이트 데이터" 체크
  4. "캐시된 이미지 및 파일" 체크
  5. "데이터 삭제" 클릭

### B. Supabase 인증 문제
- 로그인 상태에서만 문제 발생
- RLS 정책 또는 세션 문제

### C. CORS 문제
- Supabase 설정 문제

## 임시 해결책

### 방법 1: 하드 리프레시
- Ctrl + Shift + R (강력 새로고침)

### 방법 2: 브라우저 캐시 삭제
- 위 A번 참고

### 방법 3: 다른 브라우저 테스트
- Edge, Firefox 등에서도 같은 문제인지 확인
